Todo app

This is a node.js app

To see current todos add "/todos" to the end of url

To get a specific todo add "/todos/[id]" to the end of url

To add a todo, post a json object with properties text and completed to the "/todos" root

To update a todo, patch a json object with properties text and completed to the "/todos" root

To delete a todo, use delete method with a json object with properties text and completed to the "/todos" root
